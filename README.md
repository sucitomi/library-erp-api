# library-erp-api

## Description

Library ERP api.
Minimális REST API könyvek, kölcsönzők, dolgozók és kölcsönzések nyílvántartásához.
Autentikáció basic JWT token-nel.

## Installation

```bash
$ npm install
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```
